# AcademicHelper 项目总结

## 📖 项目概述

**AcademicHelper** 是一款专为 macOS 设计的学术辅助应用，旨在帮助研究人员和学生更高效地进行文献阅读、词汇学习、学术写作和知识管理。

- **项目名称**: AcademicHelper
- **平台**: macOS 14.0+
- **开发语言**: Swift 6.0 + SwiftUI
- **架构模式**: MVVM + 依赖注入 + 事件驱动
- **项目状态**: ✅ 核心功能全部完成

---

## 🎯 核心功能模块

### 1. 单词学习模块 ✅

| 功能 | 描述 | 状态 |
|------|------|------|
| 屏幕取词 | 使用 Accessibility API 捕获屏幕选中文本 | ✅ |
| 词典查询 | 集成有道/Free Dictionary API，支持离线缓存 | ✅ |
| 生词本管理 | 添加、编辑、删除、搜索单词 | ✅ |
| SM-2 复习算法 | 科学的间隔重复算法，优化记忆效果 | ✅ |
| 复习界面 | 四种评分选项（重来/困难/良好/简单） | ✅ |

**技术亮点**:
- 实现完整的 SM-2 间隔重复算法
- 双层缓存策略（内存+磁盘）
- 支持多种词典源切换

## 📝 使用说明

### 快速开始

1. **安装依赖**: 确保 macOS 14.0+
2. **配置 API 密钥**: 在设置中配置 LLM API 密钥
3. **导入文献**: 拖拽 PDF 文件到应用
4. **开始取词**: 使用快捷键 `⌘+⌥+C` 激活屏幕取词
5. **每日复习**: 使用 `⌘+R` 开始单词复习

### 快捷键列表

| 快捷键 | 功能 |
|--------|------|
| `⌘+⌥+C` | 屏幕取词 |
| `⌘+V` | 打开生词本 |
| `⌘+R` | 开始复习 |
| `⌘+⇧+Space` | 快速问答 |
| `⌘+F` | 搜索 |
| `⌘+,` | 设置 |
